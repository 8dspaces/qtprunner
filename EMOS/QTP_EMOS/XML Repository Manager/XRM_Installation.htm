Prerequisites

To be able to use the XML Repository Manager succesfully your system must meet certain preconditions. They are :-

  • Make sure that you have the MICROSOFT .NET Framework Version 1.1 installed on your computer. If you dont have it installed, you can obtain the .NET Framework installation from the following link:-
    http://www.microsoft.com/downloads/details.aspx?FamilyId=262D25E3- F589-4842-8157-034D1E7C F3A3&displaylang=en
     
  • To use the Descriptive Object Repository Parser(DOR Parser), you must have MSxml4.0 installed on your computer. If you dont have it, you may obtain it from the following link:-
    http://www.microsoft.com/downloads/details.aspx?displaylang=en&fami lyid=3144b72b-b4f2-46da-b4b6-c5d7485f2b42
     
  • If you want to use the import facility of the Repository Manager,
    then you must have the RepositoryUtil installed. The RepositoryUtil
    is found in the Quicktest Plus CD.

Installing the repository manager

To install the XML repository manager, simply run the setup.exe program. As the installation proceeds, you are prompted with various options. It is recommended that you follow the default options.

If you use the default installation, the setup program also installs the DOR Parser. The DOR parser is the library used for descriptive programming using an XML Repository. To setup Quicktest to use the XML Repository, refer the document that is installed along with the DOR Parser. The document can be found at
       <InstallDir>\DORParser\docs\DOR_Html.htm

Requesting and installing a License Key

To be able to use the Repository Manager, you must have a License Key. From your programs menu,launch the application
   Qantom -> XML Repository Manager.
Since this is the first time that you are running the application, an Invalid License dialog is first shown. This dialog has a form that needs to be filled up. Once you fill up the form and press on the 'Send' button, a license request is made on your behalf. You should then receive a license key and steps on how to install the license via email. The license key should normally be received in less than one business day.

Once you receive your license key and install it you should then be
able to use your XML Repository manager.

Good Luck and enjoy!
