Hello World Demo

EMOS Framework is a simple yet powerful environment for development of automated WinRunner® tests. This package contains the "Hello World" demo which is an example project that is normaly used as a starting point for (completely) new projects.

Getting Started

  • Make sure EMOS Framework is installed in <wrun_install_dir>\lib\EMOS_GPL directory (if not you will have to adjust paths in Scripts\startup script.
  • Set the Scripts\startup as WinRunner startup script and re-start WinRunner.
  • Run the Scripts\kickoff_new_style script in ineractive mode. Don't worry about the dialog stating that the script is read-only. Everything in EMOS Framework is read-only because we keep our files under version control. If you want to change the code, then make script file writable (and don't worry about warnings that database could not be saved).
    1. Press OK at the dialog mentioning "testset:1".
    2. A dialog saying "Hello World!" should appear.
    3. Continue and check the test results after script finished.
    4. Open and analyse Data\suite1.xls file.
    5. Open and analyse Data\table1.xls file.
    6. Open and analyse Scripts\LIB\FRM\frm_lib1_lib script.
    7. Open and analyse Scripts\DRV\drv1_lib script.
    8. Check the kickoff_new_style script that you've just run to see how all got started
    9. That's it. Now you should try FlightDemo example.
  • For other examples consult EMOS Framework home page.
