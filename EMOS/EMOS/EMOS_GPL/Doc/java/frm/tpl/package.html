This package contains the template for the typical EMOS Framework project. To activate the template do the following:
  1. copy this directory to a destination of your choice
  2. set the "Scripts/startup" script as your startup script
  3. make sure the add-in manager will show up when you restart WinRunner
  4. re-start WinRunner
  5. activate EMOS Framework add-in
  6. run "kickoff" script in interactive mode
  7. pop-up dialog showing "Hello world!" should appear
Please note that TPL is the absolutely minimal example of an EMOS Framework project. For a more sophisticate examples visit www.emos.de under menue Download;Testing.
