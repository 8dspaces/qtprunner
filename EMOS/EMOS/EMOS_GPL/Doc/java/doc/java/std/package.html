This package contains standard utility functions wich are distributed with EMOS Framework. These functions are generic in nature and can be used independently of EMOS Framework. The following naming convention has been applied: <vendor>_<package>_<name>_lib
  • vendor: indicator of code origins, e.g. emos
  • package: std (stands for "standard")
  • name: short name inidicating the nature of the library (also used as a prefix for functions contained in that particular library)
  • lib: constant indicating the compiled module
