This package contains libraries that implement custom routines for access and navigation through ddt_ tables. The following naming convention for libraries has been applied: <vendor>_<package>_<name>_lib
  • vendor: indicator of code origins, e.g. emos
  • package: ddt (stands for "data-driven test")
  • name: short name inidicating the nature of the library (also used as a prefix for functions contained in that particular library)
  • lib: constant indicating the compiled module
